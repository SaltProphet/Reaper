## Using These Documents

### For AI Coding (GitHub Copilot, Claude, etc.)

1. **Find your task** in the relevant PLAN document
2. **Copy the AI-Ready Prompt** (in code blocks)
3. **Paste to your AI assistant**
4. **Implement** following the requirements
5. **Check acceptance criteria** when done

### For Quick Tasks

1. Open the QUICK_REF document for your phase
2. Find the section for your task
3. Copy the short prompt
4. Paste and implement

### For Planning

1. Read the full PLAN document
2. Review timeline estimates
3. Check dependencies between tasks
4. Use success criteria for milestones

### For Code Review

1. Check task's acceptance criteria
2. Verify requirements met
3. Ensure follows REAPER conventions
4. Reference documentation links
